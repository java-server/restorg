Hello and welcome

This is going to grow into a tool used world wide to help cafes and restaurants in the day to day running of the business

Project lead Jay Davis is creating this tool as an add on to inventory management software and accounting software.

the tools is able to take orders, process them through to orders in the kitchen, update the accounting software, activate a countdown till ready timer and then help dispatch orders to the correct table, or call the customer to collect their order when it is complete

There will be a number of screens used in this application, it will also be able to be run on pcs or tablets

The application will not need an internet connection, but will require a LAN connection to work.
